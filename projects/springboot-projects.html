🌿 Library Management System

A full-featured Library Management System built with Spring Boot that enables librarians to manage books, members, and borrowing operations through a clean RESTful interface.

🎯 Project Goal

Provide a backend system to handle book inventory, member registration, and borrow/return transactions with proper validation and record-keeping.

⚙️ Key Features

  • Add, update, and delete book records
  • Member registration and management
  • Issue and return books with due-date tracking
  • Search books by title, author, or ISBN
  • RESTful API with Spring Boot
  • Persistent storage using MySQL / H2 database
  • Spring Data JPA for database operations

🛠 Tech Stack

Java, Spring Boot, Spring Data JPA, Hibernate, MySQL, Maven

📌 Project Type

Backend Development • REST API • Spring Boot

🎓 Student Management System

A Student Management System built with Spring Boot that allows administrators to manage student records including enrollment, courses, and academic information through a REST API.

🎯 Project Goal

Provide a robust backend for storing and retrieving student data, managing course enrollments, and performing CRUD operations on student records.

⚙️ Key Features

  • Add, update, view, and delete student records
  • Manage course registrations per student
  • Search students by name, roll number, or department
  • RESTful endpoints with proper HTTP status codes
  • Input validation and error handling
  • Spring Data JPA with MySQL / H2 database
  • Clean layered architecture (Controller → Service → Repository)

🛠 Tech Stack

Java, Spring Boot, Spring Data JPA, Hibernate, MySQL / H2, Maven

📌 Project Type

Backend Development • REST API • CRUD • Spring Boot
